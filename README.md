> 本仓库为「小X宝开源医疗社区」与「魔搭（ModelScope）」联合举办的开源医疗黑客松活动物料集合。

## 项目背景

小X宝社区是一个聚焦**罕见病与医疗健康**的开源社区，拥有多个以人体器官拟人化为核心的 IP 形象。本次黑客松旨在联合魔搭（阿里旗下 AI 开发者社区）平台，围绕医疗健康场景，鼓励参赛者开发 **Skills** 或 **MCP 工具**，并通过魔搭平台提交与展示。

### 合作方

| 角色 | 名称 | 说明 |
|------|------|------|
| 联合主办 | **魔搭 ModelScope** | 阿里旗下 AI 开发者社区，提供赛事平台与运营资源 |
| 合作方 | **KnowS** | 参与联合推广 |
| 大模型赞助 | **阶跃星辰** | 可能提供大模型 API 额度作为奖励 |

### 提交入口

- **Skill 提交**：[https://modelscope.cn/skills/create?template=custom](https://modelscope.cn/skills/create?template=custom)
- **MCP 提交**：[https://modelscope.cn/mcp/servers/create](https://modelscope.cn/mcp/servers/create)

> 提交后非反动信息即可发布至社区，赛后统一整理为赛果合集进行宣传。

## 活动沟通要点

以下内容摘自与魔搭方的沟通记录（`文档/问题.md`）及赛事文案（`文档/比赛文案.md`），记录了 6 个最优先确认的问题及回复：

### 1. 赛事详情页形式
- **形式**：采用图文形式（方便随时修改），可配合图片做宣传。
- **进度**：**主视觉与 Banner 已由徐老师和高琛同学完成**，文字部分将与 Banner 一起打包提交。后续拓展将用 agent 工具完成。

### 2. 活动时间节点（6.18-7.12 紧凑冲刺，直通 WAIC）

| 阶段 | 时间 | 说明 |
|------|------|------|
| 赛事上线 / 报名开始 | **6月18日（周四）** | 在魔搭社区完成身份绑定，加入官方交流群 |
| 选题与开发 | **6月18日 - 7月11日** | 确定医疗选题并开发，期间可随时在魔搭提交版本 |
| 作品提交截止 | **7月12日（周六）23:59** | 将作品发布至魔搭社区，并填写最终提交表单 |
| 集中评审 | **7月13日 - 7月14日（周末）** | 双轨制评分（社区热度 40% + 专家评审 60%） |
| 结果公示与 WAIC 门票发放 | **7月15日（周一）** | 公布获奖名单，Top3 发放 WAIC 门票 |
| WAIC 赛果展示 | **7月17日 - 7月20日** | Top3 获奖者于上海 WAIC 大会现场做赛果展示 |

### 3. 评审规则（双轨加权评分制）
采用 **社区热度 + 专家评审** 的双轨评分，加权总分排名决定奖项归属：
- **社区热度分 (40%)**：基于作品在魔搭社区的调用数与收藏数。
- **专家评审分 (60%)**：由医疗专家与技术大咖从**实用性、代码质量、文档完整性**三大维度打分。

### 4. 奖励体系
- **资源支持**：获胜者将赢取丰厚的云资源或 API 额度，助力项目持续演进。
- **荣誉见证**：所有完成提交的参与者均可获得官方电子参与证书。
- **品牌曝光**：优秀作品将上架 GitHub 与魔搭社区，获得全平台流量支持。
- **一等奖 (Top 3)**：1000 RMB 云资源或 API 额度 × 3 + WAIC 门票（7月15日发放）。
- **二等奖 (Top 4-10)**：500 RMB 云资源或 API 额度 × 7。
- **三等奖 (Top 10-20)**：100 RMB 云资源或 API 额度 × 10。

### 5. Skills/MCP 提交流程
- Skill 提交入口：`https://modelscope.cn/skills/create?template=custom`
- MCP 提交入口：`https://modelscope.cn/mcp/servers/create`
- 提交后非反动信息即可发布，赛后整理为合集统一宣传

### 6. 参赛者提交要求
- 视赛题设计和交付成果而定
- 魔搭目前可通过**创空间**和**灵感流**承接应用演示产物

### 7. 魔搭运营支持与口径
- 赛果回流到魔搭社区后可申请：首页/竞赛 banner、研习社置顶、社区公告置顶、公众号投稿等。
- 小程序投放需**额外申请**。
- **核心口径**：赛果回流和运营支持方式代表社区同意，形式上给社区 CMC 对齐一下。本次比赛目的之一就是和 ModelScope 共建 AI 医学生态板块。
