**Q3：云资源或 API 额度的奖励档位、具体数额和发放规则是什么？**
> **魔搭侧原复**：这部分我带回内部申请一下

**我们的最终确认**：

**Q5：参赛者提交仓库地址时，是否要求公开仓库、固定 README 模板、License、演示视频或部署链接？**
> **魔搭侧原复**：看我们的题目如何设计，并且交付的赛果是什么，如果是一个具体的应用，魔搭目前只有创空间和灵感流能承接应用演示的产物，

**我们的最终确认**：
为保证赛果质量和开源生态，我们要求参赛者提交：
1. **魔搭作品链接**（确保公开可访问，有基本 Demo）。
2. **公开代码仓库地址**（需含开源 License，推荐 Apache 2.0 或 MIT）。
3. **规范 README**（须包含：背景说明、医疗场景解决点、使用方法、部署步骤）。
